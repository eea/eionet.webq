Creating a custom control using xf:output

The xf:output value is represented as Google Map where the xf:output value represents a geolocation on the map

  1. take care that data-bf-custom-control is copied through in the XSL files
  2. check for "data-bf-custom-control" in XFControl.js and manipulate CSS classes to avoid matching of existing bf behaviours and add the value of 'data-bf-custom-control' as CSS class
  3. create a new behaviour matching the value of 'data-bf-custom-control'
