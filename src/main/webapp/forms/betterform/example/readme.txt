This directory contains XForms examples that show certain features and should work. In contrast in directory test
there are sometimes files that do not run but are kept for reference (illustrating a bug or are created for
developing a new feature).

Please note that this directory is introduced with version:
betterFORM 5.0rc1 - buildNumber:11533 - Timestamp:2012-10-30 16:05:09

Examples in this directory might not work with older builds.
